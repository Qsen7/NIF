head връща първия елемент на списък.
tail връща списъка без първия елемент.
last връща последния елемент на списък.
init връща списъка без последния елемент.
(!!) връща елемент на дадена позиция.

length връща броя елементи.
null проверява дали списъкът е празен.
elem проверява дали даден елемент се среща вътре.

filter запазва само елементите, които удовлетворяват условие.
take връща първите N елемента.
drop премахва първите N елемента.
takeWhile взема елементи, докато условието е вярно.
dropWhile премахва елементи, докато условието е вярно.

map прилага функция върху всеки елемент.
reverse обръща реда.
sort подрежда елементите във възходящ ред.

foldr и foldl комбинират елементите един по един с операция.
sum събира всички елементи.
product умножава всички елементи.
maximum връща най-големия елемент.
minimum връща най-малкия елемент.


Проверява за четно число
isEven x = x `mod` 2 == 0
main = do
  x <- getLine
  let x1 = read x :: Int
  print(isEven x1)

Да се намери хипотенузата от 2 катети
hypotenuse a b = sqrt (a^2 + b^2)
main = do
  a <- getLine
  let a1 = read a :: Float
  b <- getLine
  let b1 = read b :: Float
  print(hypotenuse a1 b1)

С bool да проверява дали двете въведени числа са положителни или отрицателни
sameSign a b = (a > 0 && b > 0) || (a < 0 && b < 0)
main = do
  a <- getLine
  let a1 = read a :: Int
  b <- getLine
  let b1 = read b :: Int
  print(sameSign a1 b1)

Фибоначи, Факториел, НОД
fib 1 = 1
fib 2 = 1
fib x = fib(x-1) + fib(x-2)

fact 0 = 1
fact x = x*fact(x-1)

nod a b = if (a==b) then a else if (a>b) then (nod (a-b) b) else (nod a (b-a))
main = do
  print(fib 6)
  print(fact 6)
  print(nod 10 15)

Функция която намира сумата на числата от 1 до n с рекурсия
sumNum 1 = 1
sumNum n = n+sumNum(n-1)
main = do
  print(sumNum 10)

Функция която приема 2 фукнции и 2 числа и изпълнява втората функция с мараметри числата, а след това подава резултата на първата функция като параметър
double n = 2*n
add a b = a+b
applyFuncs a b c d = a (b c d)
main = do
  print(applyFuncs double add 3 4)

Фукнция която изписва числата от n до 1 с рекурсия
sumNum n =
  if n == 1
    then print (1)
    else do
      print (n)
      sumNum (n - 1)
main = do
  sumNum 10

Намира лицето на правоъгълник и ако е четно, умножава на 2 и ако е нечетно - на 3
area a b =
    if (a * b `mod` 2 == 0)
        then (a * b) * 2
        else (a * b) * 3
main = do
    print (area 5 3)
